## Project Overview

This project performs comprehensive Exploratory Data Analysis (EDA) on IMDB movie data to uncover insights about:
- Movie rating trends over decades
- Genre evolution and popularity
- Factors influencing movie success
- Runtime patterns and their correlation with ratings

## Objectives

- Learn and practice data analysis using R
- Master data cleaning and transformation techniques
- Create professional data visualizations
- Build interactive dashboards
- Practice SQL queries and database management
- Develop reproducible data analysis workflows

## Tech Stack

**Programming & Analysis:**
- R 4.4+
- SQL (SQLite)

**Key Libraries:**
- tidyverse (dplyr, ggplot2, tidyr)
- plotly (interactive visualizations)
- DBI & RSQLite (database operations)
- rmarkdown (reporting)

**Visualization Tools:**
- Power BI / Tableau

**Version Control:**
- Git & GitHub
